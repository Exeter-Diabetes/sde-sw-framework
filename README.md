# Introduction

This repository contains definitions of the core diabetes dataset, including codelists and implementation rules for deriving its components, alongside code to extract these in a reproducible way. It also contains information on related projects. 

## Documentation

The [full documentation](https://exeter-diabetes.github.io/sde-sw-framework/) is in the form of a GitHub Pages static website. This site is autogenerated via a GitHub action from the relevant markdown files in this repo using the MkDocs tool.

## Contributing

Instructions on how to contribute to this repo can be found in the [CONTRIBUTING.md](CONTRIBUTING.md) document. It also contains instructions on how to build and view the documentation locally.
