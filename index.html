Data Specification

Medical code lists and Definitions

Overview

This data specification contains standardized code lists and definitions for use in research. Each dataset includes detailed descriptions of the coding rules and data sources.

The code lists are provided in multiple coding systems (SNOMED CT, ICD-10, etc.) where applicable to support diverse healthcare data sources and analytical requirements.
